# CSerDvectorOPE service (server side) implementation
This directory aims to structure computing logic (software) of CSerDvect structured in two subprojects, CSerDvectorAPIM with interface definitions and data models, and CSerDvectorOPE with business logic and complementary resources implementing the CSerDvector computing service.

## Generate both CSerDvectorAPIM and CSerDvectorOPE

mvn clean

mvn install
