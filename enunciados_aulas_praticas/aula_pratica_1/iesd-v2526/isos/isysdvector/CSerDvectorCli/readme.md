# CSerDvectorCli service client implementation
This directory aims to structure computing logic (software) of CSerDvectorCli structured in two subprojects, CSerDvectorCliAPIM with interface definitions and data models, and CSerDvectorCliOPE with business logic and complementary resources implementing the CSerDvectorCli computing service.

## Generate both CSerDvectorCliAPIM and CSerDvectorCliOPE

mvn clean

mvn install
