# # The OACI Dashboard Operations Element (OPE)
This directory aims to structure the software (computing logic) of CSerOaciDashboardOPE with business logic and complementary resources that implement the CSerOaciDashboard computing service.

## Generate implementation logic, service payload
mvn clean install

## Access CSerOaciDashboard Application, implemented by CSerOaciDashboardOPE
http://localhost:8080/generic-browser.html
